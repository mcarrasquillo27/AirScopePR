# AirScopePR

## Overview
AirScopePR is an air quality awareness system designed for Puerto Rico. It combines Virtual Reality (VR) and Augmented Reality (AR) to educate users about particulate matter, its origin, and its impact on health and the environment.

## Problem
In Puerto Rico, there is a lack of accessible and interactive information about air particulate matter. Most people are unaware of its sources, concentration levels, and health risks.

## Solution
AirScopePR provides an inmersive experience where users can:

- Explore particulate matter in a virtual laboratory (VR)
- Visualize real-time air quality data around them (AR)
- Understand sources, risks, and environmental impact
- Interact with a virtual microscope to analyze particles

## System Architecture
The system is composed of:

- External APIs (air quality + geolocation)
- Backend for data processing
- Database for storage
- Internal API for data distribution
- VR and AR modules for user interaction
