# System Architecture - AirScopePR

## Overview AirScopePR is a system designed to educate users about particle metter in Puerto Rico usign Virtual Reality (VR) and Augmented Reality (AR)

# Componets

### 1. Data Source
External APIs will provide real-time air quality data and location.

### 2. Backend
Responsile for processing, storing, and serving data.

### 3. Database
Stores processed particulate data and locations.

### 4. VR Enviroment
Allows users to explore particulate matter in a virtual lab.

### 5. AR System
Display real-time particulate data around the user.
